# 제1장. 시장조성(Market Making)의 기본 개념

## 1.1 시장조성이란?

시장조성이란 **증권사(마켓메이커, Market Maker, MM)**가 특정 파생상품에 대해 **매수호가와 매도호가를 동시에 제시**하여 시장에 유동성을 공급하는 제도입니다.

쉽게 말해, 투자자가 코스닥150 선물을 사고 싶은데 파는 사람이 없다면 거래가 안 됩니다. 이때 시장조성자가 "나는 이 가격에 팔게요(매도호가)"와 "이 가격에 살게요(매수호가)"를 항상 내놓아서 **언제든지 거래가 가능하도록** 해주는 것입니다.

## 1.2 마켓메이커가 해야 하는 일

마켓메이커는 다음을 **의무적으로** 수행해야 합니다:

1. **양방향 호가 제출**: 매수호가와 매도호가를 동시에 제출
2. **의무스프레드 준수**: 매도호가와 매수호가의 차이(스프레드)가 정해진 범위 이내여야 함
3. **의무호가수량 준수**: 최소한의 수량 이상을 호가에 제시해야 함
4. **의무이행률 달성**: 하루 중 일정 시간 이상 호가를 유지해야 함

## 1.3 마켓메이커가 받는 혜택

- **거래수수료 면제**: 시장조성 계약 상품에 대한 수수료 면제
- **거래증거금 감경**: 증거금을 적게 내도 됨
- **시장조성대가 지급**: 성과에 따라 거래소로부터 대가를 받음

# 제2장. 시장조성 의무종목

## 2.1 의무종목 개요

시장조성상품으로 **새로 선정된 코스닥150선물은 최근월물을 의무**로 하고, **주식선물은 일부 상품에 대해 차근월물도 의무**를 부여합니다.

- 주식선물 1~5라운드에서 선택된 상품
- 예: '갑'조성자가 3라운드에 'A'주식선물 선택시 'A'주식선물은 차근월물 의무가 되며, 이후 'A'주식선물을 선택한 '을'조성자는 선택 라운드와 무관하게 차근월물 의무를 부담
- 성과평가(초과의무이행도, 호가스프레드, 호가수량)는 제외

※ '24.11월 공지한 차근월물 조성상품 확대는 시장상황을 고려하여 점진적으로 추진

즉, 하루 중 최소 331.5분 이상은 매수호가와 매도호가를 동시에 내놓고 있어야 합니다.

# 제7장. 시장조성 성과평가 세부 산식

## 7.1 초과의무이행도

**개념**: 기준 이행률을 초과해서 이행한 정도를 평가

### 수식 (전/후 동일)

```
점수 = 배점 × Σ(상품, 의무종목, 일자) MAX(0, (의무이행시간 - 의무기준시간) / 초과가능시간)
       ÷ 총 의무발생일수 ÷ 의무종목 수 ÷ 시장조성상품 수
```

### 세부 설명

1. **초과 가능시간**(소수점 첫째 자리에서 반올림): 일별 의무발생시간 × (1-일중의무이행률)
2. **의무 이행시간**: 해당일에 시장조성자가 실제로 의무를 이행한 시간
3. **의무 기준시간**(소수점 첫째 자리에서 반올림): 일별 의무발생시간 × 일중의무이행률

### 계산 예시

**삼성전자 주식선물 시장조성자**

- 일별 의무발생시간: 390분 (6시간 30분)
- 일중의무이행률: 85%
- 실제 의무이행시간: 370분

**계산**:
1. 의무기준시간 = 390분 × 0.85 = 331.5분 → 332분(반올림)
2. 초과가능시간 = 390분 × (1 - 0.85) = 58.5분 → 59분(반올림)
3. 초과이행도 = MAX(0, (370 - 332) / 59) = MAX(0, 38/59) = **0.644**

## 7.2 호가스프레드

**개념**: 시장조성자가 제시한 스프레드가 의무스프레드보다 얼마나 좁은지 평가

### 수식 (전/후 동일)

```
점수 = 배점 × (1 - Σ(상품, 의무종목, 일자) MIN(1, 시장조성자평균스프레드/의무스프레드)
       ÷ 총 의무발생일수 ÷ 의무종목 수 ÷ 시장조성상품 수)
```

### 세부 설명

1. **의무스프레드**: 시장조성계약에서 정한 의무스프레드로서 주가지수선물 및 변동성지수선물의 경우에는 의무스프레드 Tick을, 주식선물, ETF선물, 주식옵션 및 주가지수옵션의 경우에는 의무스프레드 비율로 한다.
2. **시장조성자 평균스프레드**: 다음 계산식에 따라 산출한다.

### 주식옵션 특별 규정 변경

- **변경 전**: 2틱 이하인 경우 만점 부여
- **변경 후**: 폐지

### 계산 예시

**삼성전자 주식선물**

- 삼성전자 주식선물 의무스프레드: 1.5%
- 시장조성자 A의 평균스프레드: 1.2%

**계산**:
1. 스프레드 비율 = 1.2% / 1.5% = 0.8
2. MIN(1, 0.8) = 0.8
3. 1일 평균 기여도 = 1 - 0.8 = **0.2**

## 7.3 호가수량

**개념**: 의무호가수량 대비 얼마나 많은 수량을 제시했는지 평가

### 수식 (전/후 동일)

```
점수 = 배점 × Σ(상품, 의무종목, 일자) MIN(1, 평균수량/(의무수량×호가수량배수))
       ÷ 총 의무발생일수 ÷ 의무종목 수 ÷ 시장조성상품 수
```

### 세부 설명

1. **의무수량**: 시장조성계약에서 정한 최소호가수량
2. **호가수량배수**: 2
3. **평균수량**: 다음 계산식에 따라 산출한다.

### 계산 예시

**삼성전자 주식선물**

- 의무호가수량: 10계약
- 호가수량배수: 2
- 시장조성자 평균수량: 15계약

**계산**:
1. 기준수량 = 10 × 2 = 20계약
2. 수량 비율 = MIN(1, 15/20) = MIN(1, 0.75) = **0.75**

의무수량의 2배인 20계약을 제시하면 만점(1.0)을 받습니다.

## 7.4 거래량

**2026년 주요 변경**:
- 거래량(80%)과 거래대금(20%) 기준을 **병행**
- 상품 내 기여도: 60% → **40%**
- 시장 내 기여도: 40% → **60%**

### 수식 - 주식선물·ETF선물·주식옵션 (변경 전)

```
점수 = 배점 × Σ(상품, 일자) [MIN(0.6, 0.6×(시장조성자거래대금/해당시장조성상품거래대금)) +
       MIN(0.4, 0.4×(시장조성자거래대금/상품군별거래대금중앙값))]
       ÷ 총 의무발생일 수 ÷ 시장조성상품 수
```

### 수식 - 주식선물·ETF선물 (변경 후)

```
점수 = 배점 × Σ(상품, 일자) [
       0.6 × MIN(1, 시장조성자거래대금/해당시장조성상품거래대금) +
       0.4 × MIN(1, 시장조성자거래대금/상품군별거래대금중앙값)]
       ÷ 총 의무발생일수 ÷ 시장조성상품 수
```

### 수식 - 주식옵션 (변경 후)

※ 주식옵션 계산식과 통일성을 위해 실질은 동일하도록 표현만 일부 수정

```
점수 = 배점 × Σ(상품, 일자) [
       0.5 × (0.8 × MIN(1, 시장조성자거래량/해당시장조성상품거래량) + 
              0.2 × MIN(1, 시장조성자거래대금/해당시장조성상품거래대금)) +
       0.5 × MIN(1, 시장조성자거래대금/상품군별거래대금중앙값)]
       ÷ 총 의무발생일수 ÷ 시장조성상품 수
```

### 수식 - 섹터지수선물, 국내지수선물·옵션 (전/후 동일)

```
점수 = 배점 × [Σ(상품, 일자) MIN(1, 시장조성자거래량/거래소가 정하는 거래량)]
       ÷ 총 의무발생일 수 ÷ 시장조성상품 수
```

### 계산 예시

**삼성전자 주식선물 (26년 기준)**

- 시장조성자 거래량: 500계약
- 해당 상품 전체 거래량: 1,000계약
- 시장조성자 거래대금: 5억원
- 해당 상품 전체 거래대금: 10억원
- 상품군별 거래대금 중앙값: 8억원

**계산**:
1. 거래량 기여도 = MIN(1, 500/1,000) = 0.5
2. 거래대금 기여도(상품 내) = MIN(1, 5억/10억) = 0.5
3. 상품 내 기여도 = 0.8 × 0.5 + 0.2 × 0.5 = 0.4 + 0.1 = **0.5**
4. 시장 내 기여도 = MIN(1, 5억/8억) = 0.625
5. 최종 기여도 = 0.5 × 0.5 + 0.5 × 0.625 = 0.25 + 0.3125 = **0.5625**

# 제8장. 스프레드 수치 맞추는 방법 상세

## 8.1 스프레드란?

**스프레드 = 매도호가 - 매수호가**

예를 들어:
- 매수호가: 50,000원 (이 가격에 사겠다)
- 매도호가: 50,500원 (이 가격에 팔겠다)
- **스프레드: 500원**

## 8.2 주식선물의 의무스프레드 계산

주식선물의 의무스프레드는 **기초자산(주식) 가격의 일정 비율**로 정해집니다.

### 예시: 삼성전자 주식선물

**가정**:
- 삼성전자 현재 주가: 104,500원 (2025년 12월 기준 실제 데이터)
- 주식선물 의무스프레드 비율: 1.5% (예시)

**계산**:
```
의무스프레드 = 104,500원 × 1.5% = 1,567.5원
→ 호가단위(100원) 기준으로 약 1,600원
```

**올바른 호가 예시**:
- 매수호가: 104,000원
- 매도호가: 105,500원
- 스프레드: 1,500원 ✓ (1,600원 이하)

**잘못된 호가 예시**:
- 매수호가: 104,000원
- 매도호가: 106,500원
- 스프레드: 2,500원 ✗ (1,600원 초과)

## 8.3 지수선물/옵션의 의무스프레드

지수선물과 옵션은 **틱(Tick) 단위**로 스프레드를 정합니다.

**틱(Tick)이란?** 가격이 변동할 수 있는 최소 단위입니다.

### 예시: 코스닥150 선물

- 코스닥150 선물 1틱 = 0.05포인트
- 의무스프레드 = 2틱이라면

**계산**:
```
의무스프레드 = 0.05 × 2 = 0.10포인트
```

**현재 코스닥150 선물 가격이 1,153.59포인트라면**:
- 매수호가: 1,153.55
- 매도호가: 1,153.65
- 스프레드: 0.10포인트 ✓ (2틱)

## 8.4 스프레드 점수 최적화 전략

1. **의무스프레드보다 좁게 유지**: 스프레드가 좁을수록 높은 점수
2. **시장 상황 모니터링**: 변동성이 높을 때는 스프레드 관리 주의
3. **호가 갱신 빈도 조절**: 적절한 빈도로 호가를 갱신하여 좁은 스프레드 유지

# 제10장. 업무협조평가 시행방안

## 10.1 제도 개요

시장조성자가 (잠재)시장 참여자의 파생상품시장 인식 및 파생상품의 이해도를 제고하기 위한 **활동에 대한 업무협조평가를 도입**

- **목적**: 투자자 인식 개선과 시장 저변 확대를 도모
- **내용**: 파생상품 시장 접근성 개선, 상품 홍보, 투자자 교육을 위한 활동을 계획하고, 이행 여부를 확인
- **평가기간**: '26.1.9(금)~'26.11.13(금)
- **배점**: 2점 → **5점**
- **점수부여방식**: 아래 점수부여 기준에 해당하는 활동*을 수행 후 '26.11.20(금)까지 제출한 결과보고서**를 기반으로 평가

\* 아래 표 중 복수의 활동을 수행할 경우 각각에 대해 점수를 부여
\*\* 자유 양식으로 작성하되, 활동을 증빙할 수 있는 사진 자료 등을 첨부 ('주식옵션 매매기능 추가' 등은 전·후 비교 사진 자료 등을 추가)

## 10.2 투자저변 확대 및 시장 활성화 기여 관련 업무협조점수 부여 기준

# 제11장. 미니코스피200옵션 관리방안 개선

## 11.1 제재 대상

시장조성자가 제출한 **시장조성계획서상 증권거래세 면제 금액 한도를 지속적·반복적으로 초과하는 경우** 단계적 제재

## 11.2 단계별 제재방안

| 구분 | 위반 수준 | 조치 내용 | 계약 관련 |
|------|----------|----------|----------|
| **1단계** (경미한 위반) | 면제한도 초과율 10% 미만 1회 | 주의 후 초과 면세금액 추가 납입 | 계약 유지 |
| **2단계** (지속적 경미한 위반, 일시적 중대한 위반) | 면제한도 초과율 10% 미만 2회 또는 면제한도 초과율 10% 이상 1회 | 회원 경고 공문 발송, 추가 위반시 계약 해지 및 차기년도 해당상품 계약 제외 안내 | 계약 유지 |
| **3단계** (지속적 위반 등) | 경고 후 초과 | 계약 해지 및 차기년도 해당상품 계약 제외 | **계약 해지** |

\* 면제한도 초과횟수는 예탁결제원으로 일일 면제한도를 초과하여 신고한 영업일수

## 11.3 모니터링 방식

- 거래소는 월별로 예탁결제원으로부터 조성자들의 증권거래세 면제내역을 송부받아 면제한도 초과여부를 모니터링중이며,
- 거래소가 월별 모니터링을 완료한 시점에 주의/경고 조치가 가능하므로 면제한도 초과시점*과 거래소의 인지시점**이 상이

\* 모니터링 기간: 월초 ~ 월말
\*\* 인지시점: 익월 초

# 부록: 실제 데이터 기반 종합 예시

## 삼성전자 주식선물 시장조성 예시

### 기본 정보 (2025년 12월 기준)

- **삼성전자 현물 주가**: 104,500원
- **주식선물 거래단위**: 10주
- **주식선물 1계약 금액**: 104,500원 × 10주 = 1,045,000원
- **의무스프레드 비율**: 1.5% (예시)
- **일중의무이행률**: 85%

### 1. 의무스프레드 계산

```
의무스프레드 = 104,500원 × 1.5% = 1,567.5원
→ 호가단위 기준으로 약 1,600원
```

### 2. 올바른 시장조성 호가 예시

| 구분 | 호가 | 수량 |
|------|------|------|
| 매도호가 | 105,200원 | 20계약 |
| 매수호가 | 103,700원 | 20계약 |
| **스프레드** | **1,500원** | - |

✓ 스프레드 1,500원 < 의무스프레드 1,600원 → **적합**

### 3. 일중의무이행률 계산

- 거래시간: 오전 9시 ~ 오후 3시 30분 (총 390분)
- 일중의무이행률 기준: 85%
- 필요한 호가 유지 시간: 390분 × 85% = **331.5분 이상**

### 4. 성과평가 점수 계산 예시

**가정**:
- 실제 의무이행시간: 370분
- 시장조성자 평균스프레드: 1.2%
- 시장조성자 평균수량: 15계약
- 의무호가수량: 10계약
- 시장조성자 거래량: 500계약
- 해당 상품 전체 거래량: 1,000계약
- 시장조성자 거래대금: 5억원
- 해당 상품 전체 거래대금: 10억원
- 상품군별 거래대금 중앙값: 8억원

**계산**:

#### (1) 초과의무이행도
- 의무기준시간 = 390 × 0.85 = 331.5분
- 초과가능시간 = 390 × 0.15 = 58.5분
- 초과이행도 = (370 - 331.5) / 58.5 = **0.658**

#### (2) 호가스프레드
- 스프레드 비율 = 1.2% / 1.5% = 0.8
- 스프레드 점수 = 1 - 0.8 = **0.2**

#### (3) 호가수량
- 수량 비율 = 15 / (10 × 2) = 0.75
- 호가수량 점수 = **0.75**

#### (4) 거래량 (26년 기준)
- 거래량 기여도 = 500/1,000 = 0.5
- 거래대금 기여도(상품 내) = 5억/10억 = 0.5
- 상품 내 기여도 = 0.8 × 0.5 + 0.2 × 0.5 = 0.5
- 시장 내 기여도 = 5억/8억 = 0.625
- 최종 거래량 점수 = 0.5 × 0.5 + 0.5 × 0.625 = **0.5625**
